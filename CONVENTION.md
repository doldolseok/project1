# 📌 Convention

## 1. 코드 컨벤션
- 변수명: camelCase 사용 (ex: userName)
- 함수명: 동사로 시작 (ex: getUser, createPost)
- 상수: 대문자 + 언더바 (ex: MAX_COUNT)
- 들여쓰기: 2칸

## 2. 커밋 컨벤션
- feat: 기능 추가
- fix: 버그 수정
- docs: 문서 수정
- style: 코드 스타일 변경
- refactor: 코드 리팩토링

예시:
feat: 로그인 기능 추가
fix: 오류 수정

## 3. 브랜치 컨벤션
- main: 최종 코드
- develop: 개발용
- feature/기능명: 기능 개발

예시:
feature/login
feature/signup
